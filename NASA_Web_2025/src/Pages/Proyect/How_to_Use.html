Exoplanet Detection Model

How to Use?

Analyze Data

Access the Home Project page to start the analysis.

Select the 'Analysis' option to begin.

Fill in the required fields with the light curve information.

Upload the light curve image of the star.

Click the 'Submit' button. The CNN model will return whether it is an exoplanet or not.

Propose Model Improvements

Select the 'Examine' button to upload a CSV file with improvement proposals.

Click 'Send' to submit your proposals to improve the model.

Adjust Parameters

Access the Home Project page again.

Select 'Tweak The Model' to adjust parameters as required.

Click 'Send'. The changes will be automatically saved.
